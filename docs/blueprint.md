# **App Name**: Calm Journal

## Core Features:

- Daily Entry: Users can create a new journal entry for each day.
- Emotion Tracking: Allow users to tag emotions such as happiness, sadness, anger in their journal entries.
- Entry Display: Present entries in a chronologically ordered and easily readable format.
- AI Prompt Generation: Generative AI suggests daily writing prompts for each journal entry to encourage journaling. The prompt is considered to be a "tool" for encouraging journaling.

## Style Guidelines:

- Primary color: Rose (#C07A84) to evoke feelings of calm and comfort.
- Background color: Blush (#F6EBEF) for a soft and gentle backdrop.
- Accent color: Cream (#FFF7F8) to highlight important elements.
- Headline font: 'Playfair' serif for a elegant, readable titles.
- Body font: 'Inter' sans-serif for a modern, clean, readable paragraphs.
- Utilize generous spacing and rounded buttons for a soft, approachable feel.
- Employ soft shadows to add depth and a tactile quality to the UI elements.
